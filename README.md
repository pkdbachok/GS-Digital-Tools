# PKD Bachok One Stop Centre — WhatsApp Share Link

## Files
- `index.html` — Open Graph metadata and automatic redirect
- `preview.jpg` — WhatsApp preview image, 1200 × 630 pixels

## Setup

1. Create a public GitHub repository named:

   `pkd-bachok-one-stop-centre`

2. Upload `index.html` and `preview.jpg` to the root of the repository.

3. Open `index.html` and replace every occurrence of:

   `YOUR_GITHUB_USERNAME`

   with your actual GitHub username.

4. Commit the change.

5. In the repository, open:

   `Settings` → `Pages`

6. Under **Build and deployment**, select:

   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`

7. Save and wait for GitHub Pages to publish.

Your shareable URL will be:

`https://YOUR_GITHUB_USERNAME.github.io/pkd-bachok-one-stop-centre/`

When shared in WhatsApp, the GitHub Pages URL supplies the custom title,
description, and image. When opened, it redirects to:

`https://sites.google.com/moh.gov.my/pkdbachok/home-page`

## Refreshing an old WhatsApp preview

WhatsApp can cache previews. After changing the image or metadata, share a
versioned URL once, for example:

`https://YOUR_GITHUB_USERNAME.github.io/pkd-bachok-one-stop-centre/?v=2`
